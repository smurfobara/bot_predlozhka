Установить нужные библиотеки можно с помощью команды pip install -r requirements.txt (находясь в папке с проектом)
В файле config в поле TOKEN установить токен бота, созданный в @BotFather
В файле config в поле admin установить установить id аккаунта админа

Запускать бот можно с помощью команды - python .\run находясь в папке с проектом
Останавливать с помощью Ctrl + C
